Tipos de nota

Aula Destinada

Enviar Nota como...

Particular

Cuando se selecciona este tipo de nota, se comunica algo sobre un alumno/hijo especifico: Alumno Referido. Al tener un Alumno referido en la Nota, el sistema ya sabe cual sera el Aula destinada.

Con aula destinada

Cuando se selecciona esta opcion, la nota es enviada al Aula deseada, es decir, el sistema automaticamente envia la nota a los Destinatarios correspondientes. Los destinatarios varian acorde a si la nota esta siendo enviada por un Padre, un Docente o un Directivo.

El sistema ha detectado que su usuario tiene asignado mas de un Rol. Esto puede ocurrir cuando un mismo Usuario actua como Padre y Docente, o como Padre y Directivo a la vez. Debe seleccionar la opcion correspondiente acorde a como desea actuar. Ejemplo, si quiere enviar una Nota Particular sobre uno de sus hijos, o enviar una Nota generica al personal de la institucion, seleccione la opcion 'Padre'. Si en cambio desea enviar una Nota generica a alguna de las aulas que Ud. tiene a cargo, seleccione Docente (en caso de tener ese Rol asignado). Sino, seleccione 'Directivo'.


Generica

Cuando se selecciona este tipo de nota, se comunica algo generico, que puede o no estar directamente relacionado a un Alumno.

Sin aula destinada

Cuando se selecciona esta opcion, la nota no es enviada a un Aula determinada, sino que es el Emisor quien debe elegir a que Destinatarios correspondientes se debe enviar la nota. Los destinatarios disponibles para su eleccion varian acorde a si la nota esta siendo enviada por un Padre, un Docente o un Directivo.
